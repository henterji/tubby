## Getting Started

Install node.
```
https://nodejs.org/en/
```

Install cnpm.
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

Install dependency packages.
```
cnpm install
```

Serve frontend app by webpack-dev-server.
```
cnpm start
open http://localhost:4200
```

Testing.
```
cnpm test
```

Mock data.
```
npm run mock
```

Production build.
```
npm run build
npm run server:prod
open http://localhost:4200
```
